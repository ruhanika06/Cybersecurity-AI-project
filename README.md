AI-Based Intrusion Detection System
Overview

This project implements a Machine Learning–based Intrusion Detection System (IDS) to detect malicious network activities. The system analyzes network traffic data and classifies it as normal or intrusive using supervised learning algorithms. The project demonstrates how AI and data analysis can help strengthen cybersecurity systems.

Features

1. End-to-end Machine Learning pipeline

2. Data preprocessing and feature selection

3. Multiple ML models for intrusion detection

4. Model performance comparison

5. Visualization of results using graphs

6. Modular and easy-to-understand implementation

Technologies Used

1. Python

2. Scikit-Learn

3. Pandas

4. NumPy

5. Matplotlib

Machine Learning Models Implemented

1. Logistic Regression

2. Random Forest

3. Support Vector Machine (SVM)

Project Workflow

1. Data collection and preprocessing

2. Feature selection and dataset preparation

3. Model training using supervised learning algorithms

4. Model evaluation and comparison

5. Visualization of performance metrics

Results

The models were evaluated based on accuracy and detection performance. Visualizations were generated to compare the effectiveness of different algorithms in identifying network intrusions.

Installation

1. Clone the repository

git clone https://github.com/ruhanika06/AI-Intrusion-Detection-System.git

2. Navigate to the project folder

cd AI-Intrusion-Detection-System

3. Install required libraries

pip install pandas numpy scikit-learn matplotlib

4. Run the project

python main.py
